# Cosmic Distance Ladder Project

## Objective
Study how astronomers measure distances in the universe using:

- Stellar Parallax (Gaia data)
- Cepheid Variable Stars (OGLE data)
- Type Ia Supernovae (Pantheon+)
- Hubble’s Law

## Methods
- Data cleaning using Pandas
- Scientific visualization using Matplotlib
- Regression analysis for Hubble constant
- Logarithmic scaling for astrophysical relations

## Key Results
- Estimated distances to stars and galaxies
- Constructed Cepheid Period–Luminosity relation
- Derived Hubble expansion relation

## Tools
Python, NumPy, Pandas, Matplotlib, Jupyter Notebook

## Data Sources
- ESA Gaia Archive
- OGLE Catalog
- Pantheon+ Supernova dataset
